Pig Dice Game

This was a project I worked that played a game called Pig Dice. We used javascript and jquery for the funtionality of the game, and css to display it all. I learned a lot during this project.

Roman Numeral Converter

This project was one I worked on when learning what the steps to testing and building up code were. We learned how to set up the tests and make sure our project was doing what it needs to do. Even though I remember running into a lot of struggles with this one, it was very fun and satisfying to finally break through and have it functional in the end.

Galactic Age Calculator

This was a project I worked on when learning Webpack and Node. It calculates what your age would be on other planets. It was crazy to begin to see how many usful tools there are for coding. I look forward to working with these tools and many more as I progress.
